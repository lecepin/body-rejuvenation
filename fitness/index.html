# 物理疗法

从进化角度来看，肌肉属于消耗品。如果长时间不使用某些肌肉，或者使用强度很低，那么身体就会将这些肌肉进行弱化处理，除非你再去刺激使用它。

所以，从这个角度，要保持长期的身体素质，就要不断的去使用身体的肌肉。

这里多关注一下，身体中下部的锻炼：

  • 腹肌
  • 背肌
  • 腰肌
  • 臀肌
  • 内收肌
  • 四头肌
  • 屈大腿肌
  • 小腿肚肌
